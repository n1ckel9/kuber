# Водовоз — план запуска и монетизации

Двусторонний маркетплейс спецтехники. Старт: **Якутск**, якорная услуга — **доставка воды**
(регулярная потребность частных домов без водопровода → повторные заказы).

Главные риски: «курица и яйцо» (нет исполнителей ↔ неинтересно заказчикам) и холодный старт.
Весь план — про ликвидность: чтобы **каждая заявка закрывалась**.

---

## Принцип
Один город, одна услуга, первые сделки сводим вручную. Не распыляться.

---

## Фаза 0. Подготовка (1–2 недели)
- [ ] Город: Якутск. Услуга: вода.
- [ ] Оффер в одну фразу: «Вода с доставкой за 1 час — выбери по цене и рейтингу».
- [ ] Каналы: Telegram-канал, группа VK, WhatsApp, карточка в 2ГИС/Яндекс.
- [ ] Заявка за 30 секунд, минимум трения (позже — вход по телефону/SMS).
- [ ] На старте всё бесплатно для всех.

## Фаза 1. Набрать ПРЕДЛОЖЕНИЕ (1–3 недели)
- [ ] Найти 10–20 водовозов: Avito/Юла, 2ГИС, городские чаты, сарафан.
- [ ] Лично завести им аккаунты (concierge): услуги, контакт.
- [ ] Оффер исполнителю: «Бесплатно даю заказы, ничего не платишь».
- [ ] Цель: 5–10 живых исполнителей, готовых брать заказы сегодня.

## Фаза 2. Запустить СПРОС + ликвидность (параллельно)
- [ ] Заказчики из районных чатов, групп СНТ/ЖК, «Подслушано», барахолок, листовок в частном секторе.
- [ ] Первые 50–100 заказов вести вручную: не пришёл отклик за 10 мин — звонить исполнителю.
- [ ] **Метрика №1 — fill rate >80%** (доля закрытых заявок). Пока ниже — не расширяться.

---

## Первые 2 недели после запуска (чек-лист)
- [ ] Ежедневно: заявок / закрыто / время до 1-го отклика.
- [ ] 5–10 живых интервью с обеими сторонами после сделки.
- [ ] Топ-3 проблемы — чинить каждую неделю.
- [ ] Просить отзыв+рейтинг после каждой сделки.
- [ ] Лог: заявка → закрыта/нет → почему (это бэклог).
- [ ] Реферальная петля: «приведи водовоза/соседа».

---

## Каналы привлечения (регион РФ)
**Сейчас, дёшево:** районные Telegram/WhatsApp-чаты, паблики VK, «Подслушано», барахолки;
2ГИС/Яндекс.Карты; Avito; офлайн — наклейки на машинах, листовки в частном секторе,
объявления в строймагазинах/автомойках/правлениях СНТ; сарафан и рефералы.
**Когда пойдёт трафик:** таргет VK по гео, посевы; партнёрства с УК, СНТ, стройбазами, ЖКХ.

---

## Как НЕ спугнуть
1. Заказчику — всегда бесплатно.
2. Монетизация — только после ликвидности.
3. Для исполнителей при включении платы:
   - грандфатеринг ранних (надолго бесплатно);
   - стартовые бонусы (первые отклики бесплатно);
   - цена ≪ ценности (отклик ~1–3% от заработка с заказа);
   - **оплата за результат**, а не «за воздух».
4. Без спама. Уведомления только по делу.
5. Модерация, реакция на жалобы, видимый рейтинг.

---

## Монетизация — когда включать (триггеры)
Не раньше, чем: **≥15–20 заказов/день**, **fill rate >80%**, исполнители просят больше заказов
(конкуренция за лиды). Тогда доступ к заказам стал ценным.

Порядок включения (мягко):
1. **Баланс исполнителя + плата за отклик** (списание за результат). ← плумбинг уже готов, выключен флагом.
2. Подписка («безлимит откликов за ~990 ₽/мес»).
3. Продвижение (поднять заявку/анкету, бейдж «Проверенный»).
4. Позже — комиссия/встроенная оплата, доп. сервисы.

Заказчику — без изменений.

### Технический статус монетизации
- Кошелёк исполнителя и транзакции — реализованы (`accounts.balance`, таблица `transactions`).
- Плата за отклик управляется конфигом сервера:
  - `BID_FEE` — стоимость отклика в ₽ (по умолчанию `0` = бесплатно/выключено);
  - `SIGNUP_BONUS` — бонус новому исполнителю в ₽ (по умолчанию `0`).
- Пополнение — заглушка `POST /api/wallet/topup` (мгновенно зачисляет). Перед боевым
  запуском подключить ЮKassa/CloudPayments вебхуком вместо заглушки + чеки 54-ФЗ.
- Включение монетизации = выставить `BID_FEE` (и `SIGNUP_BONUS`) в окружении сервера.

---

## Приборная панель (еженедельно)
| Метрика | Цель |
|---|---|
| Заявок/день | растёт |
| Fill rate | >80% |
| Время до 1-го отклика | <15 мин |
| Повторные заказы (вода) | растёт |
| Активные исполнители | ≥ спрос |
| Рейтинг/жалобы | стабильно высокий |

Fill rate падает → тормозить спрос, добирать исполнителей (и наоборот).

---

## Антипаттерны
- ❌ Много городов/услуг сразу.
- ❌ Монетизация до ликвидности.
- ❌ Брать деньги с заказчиков.
- ❌ Ждать, что пользователи «сами разберутся».
- ❌ Гнать трафик в дырявое ведро (баги/низкий fill rate).

---

**TL;DR:** Якутск + вода → 5–10 водовозов вручную → заказчики из местных чатов →
первые 50–100 заявок закрыть руками до fill rate >80% → рост сарафаном/рефералами →
включить плату за отклик/подписку, когда исполнители конкурируют за заказы
(заказчикам бесплатно, ранним — бонусы).
