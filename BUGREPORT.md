# Багрепорт «Кубер» (ревью прототипа)

Статус: прототип работает. Ниже — реальные баги и риски перед боевым запуском,
по приоритету. Часть пунктов уже исправлена в этом проходе (помечено ✅).

## 🔴 Критичные — ✅ ИСПРАВЛЕНО

1. **Rate-limiting.** ✅ Добавлен in-memory лимит по IP+маршруту: `request-otp` 5/мин,
   `verify-otp` 15/мин, `login` 20/мин, `register` 10/мин, `geocode*` 60–90/мин.
2. **OTP-брутфорс.** ✅ Лимит 5 попыток → код сгорает (429).
3. **Открытый geocode-прокси.** ✅ `/api/geocode*` теперь требует токен (`requireAuth`) + rate-limit.
4. **Бессрочные токены.** ✅ Срок жизни сессии 30 дней; просроченные удаляются.
5. **CORS `*`.** ✅ Управляется `CORS_ORIGIN` (по умолчанию `*` для dev; в проде задать домен).
6. **Угон аккаунта по телефону.** ✅ Вход по OTP находит только phone-аккаунты
   (`phone%@kuber.local`), email-аккаунты по номеру не захватываются.
7. **Возврат платы за отклик.** ✅ Рефанд при отклонении, удалении заявки и
   проигрыше (когда `BID_FEE`/`%` включены).
8. **`PATCH /api/config`.** ✅ Теперь `requireAdmin` (менять тариф может только админ).

> Остаточные улучшения по теме (на потом): 6-значный OTP + лимит частоты запроса кода;
> кэш геокодера; ротация токенов; `express-rate-limit`/Redis для нескольких инстансов.

## 🟠 Высокие

9. **Отрицательные цены.** ✅ Заказ/ставка/расписание требуют `price > 0`.
10. **Прод-инфраструктура.** ⏳ Сейчас SQLite + локалка. Для пилота ок; для роста —
    Postgres + хостинг + HTTPS + бэкапы. (Инфраструктура, не код — шаг к запуску.)
11. **Нет автотестов и мониторинга.** ✅ Добавлен смоук-тест `npm test` (18 проверок:
    auth, заказы, lifecycle, оплата, рефанд, rate-limit), логирование ошибок сервера
    и опциональный лог запросов (`LOG=1`). Полноценный Sentry — позже.
    + ✅ Лимиты длины полей (адрес/задача/имя) — защита от мусора и раздувания БД.

## 🟡 Средние

12. **Polling вместо WebSocket** (чат 5с, лента/уведомления 12с). ⏳ Архитектурное —
    для прода перейти на WebSocket/SSE. Для пилота приемлемо.
13. **Push только в dev/prod-build на устройстве.** ⏳ Ограничение Expo Go SDK 54;
    плумбинг готов, нужен билд (это и делаем через EAS).
14. **Карта/геокодер на бесплатных OSM/Nominatim** — fair-use лимиты. ⏳ Для прода —
    OpenFreeMap/MapTiler + self-host Nominatim. (Сейчас geocode защищён авторизацией и rate-limit.)
15. **Уведомления растут без очистки.** ✅ Храним последние 100 на аккаунт.
16. **Генератор расписаний.** ✅ Добавлен guard от наложения тиков. (Мульти-инстанс
    всё ещё требует внешней блокировки — актуально только при горизонтальном масштабировании.)
